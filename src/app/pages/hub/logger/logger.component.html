STEP 1 — SAVE THE EXPORT JSON

Click EXPORT BUNDLE above to download the encrypted bundle JSON.

STEP 2 — DECRYPT VIA CLI

node scripts/decrypt_bundle.mjs --bundle export.json --private-key private.pem

Requires Node 18+. Exit code 0 = decrypted, 2 = nothing decryptable.

STEP 3 — OR USE THE SDK

import { readFileSync } from 'node:fs';
import { decryptExportBundle } from '@obrioxia/client-decryptor';

const result = await decryptExportBundle({
  privateKeyPem: readFileSync('private.pem', 'utf-8'),
  bundleJson: JSON.parse(readFileSync('export.json', 'utf-8')),
});

console.log(result.decrypted_fields);
console.log(result.undecryptable_fields);

Use repo-local SDK in client_sdk/node if not published to npm. Zero dependencies, Node 18+.

Output fields: decrypted_fields, undecryptable_fields, bundle_fingerprint (SHA-256), metadata (decision_id, key_epoch).
