Platform Features

Enterprise-grade controls for the AI age.

Tamper-Evident Decision Ledger

A tamper-evident, append-only database that serves as the single source of truth for your AI operations.

  • ► SHA-256 hash chaining ensures historical records cannot be silently altered.
  • ► Canonical JSON serialisation supports deterministic verification.
  • ► API-first ingestion designed for agentic workflows.

Chain Verifier

A cryptographic verification tool that allows auditors, regulators, or customers to validate the integrity of your logs independently.

  • ► Zero-trust architecture: don't trust the database, verify the math.
  • ► Instant integrity checks via the Obrioxia Hub.
  • ► Any modification — even a single byte — breaks the chain and is detected on verification.
  • ► Multi-mode lookup: verify by hash, decision ID, or 0x-prefixed hash — the endpoint normalises automatically.
  • ► Full-chain integrity verification in a single API call.

Evidence Packs (JSON + PDF)

Downloadable, portable proof packages containing the raw log data, the cryptographic proof, and a human-readable summary.

  • ► PDF audit certificates for non-technical stakeholders.
  • ► Machine-readable JSON for automated auditing.
  • ► Portable proof packages for sharing and external review.
  • ► Ownership-verified: PDF generation requires proof you created the record.

Privacy-Aware Immutability (Crypto-Shredding)

Our write-only architecture allows you to remove access to sensitive user data to support GDPR Article 17 obligations, without breaking the hash chain that records the event.

  • ► Encrypts PII before hashing using per-event write-only keys.
  • ► "Shredding" a key renders the PII mathematically unrecoverable.
  • ► The audit metadata remains intact, proving the event occurred while respecting privacy.
  • ► Dual shred path: v2 records have their wrapped encryption key destroyed; legacy records have PII replaced with a shredded marker.

Obrioxia Hub

The central command center for managing your audit streams, verifying chains, and executing GDPR erasure requests.

  • ► Real-time Event Logger for system testing and integration.
  • ► Built-in Chain Verifier for instant health checks.
  • ► GDPR Admin panel for authorized crypto-shredding actions.
  • ► Analytics: API endpoints for confidence scores, trends, and anomaly detection.
  • ► Entitlement dashboard: view your tier, event usage, and remaining quota.
  • ► Evidence PDF download with ownership verification.

Integrations & SDKs

Drop-in client libraries for your language of choice. Zero-config auth handling, typed responses, and built-in timeout support.

  • ► Node.js / TypeScript — native fetch, zero dependencies, Node 18+. @obrioxia/sdk
  • ► Python — httpx-based, context-manager support. obrioxia-sdk
  • ► Both SDKs support demo-key and Bearer-token auth, incident logging, chain verification, and health checks.
