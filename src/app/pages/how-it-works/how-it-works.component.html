How It Works

Think of Obrioxia as a forensic flight recorder for your AI fleet. It creates a tamper-evident record of every decision — the "digital alibi" you need when liability is in question.

1

Connect

Sign up, choose a plan, and activate your subscription. Then generate your API key in the Obrioxia Hub — the onboarding wizard walks you through setup in under two minutes.

2

Ingest

Your AI sends decision events to the Obrioxia API — inputs, outputs, confidence scores, and metadata. Sensitive fields are sealed on arrival with write-only protection so the production system can never read them back.

3

Seal

Each record is cryptographically locked to the one before it, creating an unbroken chain. If anyone modifies or deletes a past record, the mathematical link breaks — and the Verifier will flag it instantly.

4

Verify

Run the Chain Verifier to prove that no record has been tampered with. Download forensic audit certificates (PDF) for regulators, or verify programmatically via the API.

5

Erase

When a user requests data deletion, use the Crypto-Shredder to destroy access to their personal data. The audit record stays in the chain — proving the event happened — but the sensitive details become permanently irrecoverable.
